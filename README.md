# Farm Management System

This project was developed as part of a 1-week practical assignment.

## Features
- Farmer registration & profiles
- Supplier product management
- Order placement & status tracking
- User roles (Admin, Farmer, Supplier, Field Officer)
- Dashboard summaries
- PDF report export

## Technologies Used
- Python (Django)
- HTML, CSS, Bootstrap
- SQLite
- ReportLab (PDF)

## How to Run
1. Clone the project
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run migrations:
   python manage.py migrate
5. Start server:
   python manage.py runserver

## Login Roles
- Admin
- Supplier
- Farmer
- Field Officer
